PJSK SUS Chart Audio Path Fixer

Quickly fixes the audio path of SUS ( Sliding Universal Score )charts for use in a simulator like Umiguri. Designed to be used with PjskSUSPatcher (although you can use it with other stuff if you have a specific use case).

Add the zip file containing the .sus charts and audio files and the tool will pick it up.

All operations are handled locally, nothing is uploaded
